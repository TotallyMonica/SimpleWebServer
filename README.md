# Simple Web Server

## Prerequisites:
[Python 3](https://www.python.org/downloads/)

## Usage:

```bat
cd C:\path\to\downloaded\script
py simple_web_server.py
```

Then access the web server on your computer or any other device on your network with the provided hostname.
